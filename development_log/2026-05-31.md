### 技术发现

1. **MotionSimTool 委托模式**：MotionSimTool 内部通过 `from .motion import ...` 委托给 motion.py 的辅助函数，保持 simulation.py 不臃肿，同时保留 `motion_sim` 这个统一的入口工具名
2. **OpenFOAM 跨平台执行**：通过 `_find_openfoam()` 自动检测运行环境（WSL/Docker/原生），`_run_openfoam_command()` 统一封装命令执行，上层代码无需关心底层平台
3. **WSL 路径转换**：`_win_to_wsl_path()` 将 Windows 路径（`C:\Users\...`）转为 WSL 路径（`/mnt/c/Users/...`），这是在 Windows 上通过 WSL 运行 OpenFOAM 的关键
4. **CFD case 文件模板化**：`_build_cfd_case()` 用 Python f-string 生成所有 OpenFOAM 配置文件，支持参数化流体属性、边界条件、湍流模型
5. **优雅降级**：无 FreeCAD 时 motion tools 返回安装提示；无 OpenFOAM 时 CFD tools 返回安装选项（WSL/Docker/原生），不崩溃
